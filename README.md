# The collection of Python calculators

### Standard calculator 
The Standard calculator is a super simple Python calculator,  based on starting out with Python 5th ed.

Go to the Standard calculator folder and go to Standard Calculator, which is the current version, which adds more clearity to the calculations. 
#### Distance-price calculator (Is a part fo the standard calculator family of programs)
There are also the experimental Distance-price folder. Distance-price calculator is a project based on the idea of calculating the price of taking the train for point a to point b. 

### Measurement calculator 
This program just converts ml to dl and dl to l.
